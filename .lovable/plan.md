

# Add Google OAuth Secrets

## What to do

You need to add two secrets to power the GSC integration. In Lovable:

1. Click **Cloud** (in the top navigation bar above the preview)
2. Go to **Secrets**
3. Add these two secrets:

| Secret Name | Where to find the value |
|---|---|
| **GOOGLE_CLIENT_ID** | Google Cloud Console → APIs & Services → Credentials → your OAuth 2.0 Client ID → copy the **Client ID** |
| **GOOGLE_CLIENT_SECRET** | Same page → copy the **Client secret** |

Once you've added both secrets, let me know and I'll test the GSC connection end-to-end.
